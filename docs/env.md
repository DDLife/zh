# 环境

## steam

### 配置

- 显示 URL: `Settings` -> `Interface` -> `Show Steam URL address bar when available`

### steam 打开链接

在浏览器中输入:

steam://openurl/`[URL]`

其中, `[URL]` 为链接

---

另一种方法: [^reddit steam url]

1. 在浏览器中输入

   [steam://open/console](steam://open/console)

2. 输入命令

   open `[URL]`

   其中, `[URL]` 为链接
