## 须知

- mod 订阅 [^mod subscribe] : steam 和 [Klei](https://forums.kleientertainment.com/files/)
- mod 本地文件夹 [^mod local] : `steam\steamapps\workshop\content\322330` 或者 `饥荒文件夹的mods文件夹`  
  mod v1 版本会在 mods 目录下面, v2 以上会在 steam/content 里面
